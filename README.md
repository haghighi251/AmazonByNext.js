<hr>

## Install

###### To install the dependencies and start the app, run the following commands:
## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```
